The overflow property

Overflow hidden:

When we use overflow as visible .The overflow is not clipped it renders outside the elements box this is default one The overflow property works only for block elements with a specified height.

Overflow scroll:

When we use overflow as visible .The overflow is not clipped it renders outside the elements box this is default one The overflow property works only for block elements with a specified height.

Overflow auto:

When we use overflow as auto the overflow is clipped and a scrollbar is added to see the rest of the content this is default one The overflow property works only for block elements with a specified height.

Overflow visible:

When we use overflow as visible .The overflow is not clipped it renders outside the elements box this is default one The overflow property works only for block elements with a specified height.
